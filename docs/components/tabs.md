# Tabs 标签页

标签切换组件，平滑动画

## 引入

```html
<link rel="stylesheet" href="/tokens/tokens.css">
<!-- 复制下面的组件代码 -->
```

## 变体

| 变体 | 用途 |
|------|------|
| `default` | 标准标签页，底部边框指示器 |
| `pills` | 药丸样式标签 |
| `underline` | 下划线指示器样式 |
