# Navbar 导航栏

响应式导航栏，支持透明和毛玻璃变体

## 引入

```html
<link rel="stylesheet" href="/tokens/tokens.css">
<!-- 复制下面的组件代码 -->
```

## 变体

| 变体 | 用途 |
|------|------|
| `default` | 标准导航栏，白色背景 |
| `transparent` | 透明背景，适合放在 Hero 上方 |
| `glass` | 毛玻璃效果，半透明背景模糊 |
