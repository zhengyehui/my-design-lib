# Alert 提示

提示信息框，四种语义颜色

## 引入

```html
<link rel="stylesheet" href="/tokens/tokens.css">

```

## 变体

| 变体 | 用途 |
|------|------|
| `info` | 信息提示，蓝色 |
| `success` | 成功提示，绿色 |
| `warning` | 警告提示，黄色 |
| `danger` | 危险提示，红色 |
