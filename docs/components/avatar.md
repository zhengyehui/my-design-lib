# Avatar 头像

用户头像组件，支持图片和文字

## 引入

```html
<link rel="stylesheet" href="/tokens/tokens.css">
<!-- 复制下面的组件代码 -->
```

## 变体

| 变体 | 用途 |
|------|------|
| `default` | 圆角矩形头像 |
| `circle` | 圆形头像 |
| `square` | 方形头像 |
