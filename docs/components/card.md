# Card 卡片

内容容器组件，用于展示文章、产品、功能等信息块。

## 变体

| 变体 | 用途 |
|------|------|
| `default` | 标准卡片，带边框和悬浮阴影 |
| `flat` | 无阴影，适合密集布局 |
| `glass` | 玻璃拟态效果，适合叠加在背景上 |
| `gradient` | 渐变背景，适合特色内容 |

## 结构

- `.card__image` — 顶部图片（16:9）
- `.card__body` — 内容区
- `.card__tag` — 分类标签
- `.card__title` — 标题
- `.card__desc` — 描述
- `.card__footer` — 底部操作区
