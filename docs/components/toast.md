# Toast 通知

右上角浮出通知弹窗

## 引入

```html
<link rel="stylesheet" href="/tokens/tokens.css">
<!-- 复制下面的组件代码 -->
```

## 变体

| 变体 | 用途 |
|------|------|
| `default` | 默认通知 |
| `success` | 成功通知，绿色 |
| `error` | 错误通知，红色 |
| `loading` | 加载中通知，旋转图标 |
