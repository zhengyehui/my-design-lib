# Spacing 间距

基于 4px 基础单位的间距系统。

| 变量 | 值 | 倍数 |
|------|-----|------|
| `--space-1` | 4px | 1x |
| `--space-2` | 8px | 2x |
| `--space-3` | 12px | 3x |
| `--space-4` | 16px | 4x |
| `--space-6` | 24px | 6x |
| `--space-8` | 32px | 8x |
| `--space-10` | 40px | 10x |
| `--space-12` | 48px | 12x |
| `--space-16` | 64px | 16x |
| `--space-20` | 80px | 20x |
| `--space-24` | 96px | 24x |

## 使用原则

- **组件内间距**：用 `--space-2` 到 `--space-4`
- **组件间间距**：用 `--space-4` 到 `--space-8`
- **区块间距**：用 `--space-12` 到 `--space-24`
